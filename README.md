# CodeForge

CodeForge is a modern coding practice platform inspired by LeetCode, designed to help students and developers improve problem-solving skills through an interactive and visually appealing interface.

# Features

## Coding Challenges
- Practice coding problems with different difficulty levels
- Supports:
  - Python 3
  - Java
  - C++
  - JavaScript

## Difficulty Filtering
Users can filter coding problems by:
- Easy
- Medium
- Hard

## AI Review System
- AI-powered code review section
- Provides feedback and improvement suggestions
- Helps users understand mistakes and optimize solutions

## Export Progress Report
- Users can export their coding progress reports
- Tracks solved problems and performance

## User Authentication
- Remembers user login sessions
- Persistent login functionality for better user experience

## Daily Streak System
- Tracks consecutive login days
- Encourages users to practice consistently
- Displays streak count for motivation

## Avatar Customization
- Users can edit profile avatars
- Supports:
  - Male avatars
  - Female avatars
- Personalized profile experience

## Company Preparation Briefings
- Displays small briefings about company expectations
- Helps users understand:
  - Required skills
  - Interview focus areas
  - Common topics asked by companies

## Modern UI/UX
- Responsive dark-themed interface
- Clean and user-friendly design
- Optimized for desktop and mobile devices
